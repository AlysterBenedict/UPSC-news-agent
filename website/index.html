Stateful LangGraph Orchestration

The entire pipeline runs as an 11-node stateful directed acyclic graph. Every transition is managed via a shared TypedDict state, with atomic SQLite checkpointing after each node execution.

  • Resume from any checkpoint after crash or power loss
  • Pydantic v2 strict schema validation at every stage
  • ThreadPoolExecutor parallel LLM inference for verification
  • Deterministic fallback heuristics backstop every ML node
